# S5 各产品类型额外覆盖项

本文件供 SKILL.md S5 阶段引用。当产品类型已确认后，除 S5 通用产出外，必须按类型补齐以下额外覆盖项。

## B2B 必须额外覆盖

- 角色权限矩阵
- 审批 / 协作流程
- 数据来源和集成系统
- 合规、安全、审计或组织落地约束

## 平台型必须额外覆盖

- 各角色（供方 / 需方 / 平台运营）独立功能视角
- 供需匹配规则和排序策略
- 平台佣金 / 抽成 / 结算规则
- 各角色间的数据可见边界和交互规则

## 内部工具必须额外覆盖

- 操作效率指标（完成任务所需步骤数、耗时）
- 与现有系统（CRM、ERP、工单系统等）的集成接口和约束
- 培训成本和上手难度评估
